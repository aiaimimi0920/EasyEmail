# cloudflare_temp_email / frontend

## Toolchain

- Node.js `20.19+`
- Corepack enabled
- `pnpm` via the repo's pinned `packageManager`

## Project Setup

```powershell
corepack pnpm install
```

## Development

```powershell
corepack pnpm dev
```

## Validation

```powershell
corepack pnpm test
corepack pnpm build
```
